Parameters:

nbr_selectedPinsNumber of pins that were specified by the user
print_to_screenThis specifies if some extra information is to be displayed in the user screen. It has no other purpose than debugging your code. If print_to_screen == 1, data will be displayed on the terminal window; otherwise it will not.
pinDescription[]A constant array containing the description of each pin
pinIDID of pin we want to pulse
delayDelay in seconds that will define the interval between signal states
